Features
  • Built by humans
  • Fully open source.
  • Great replacement for speed sheet users.
  • Great for classrooms.
  • Store your projects in one place.
  • Export final project to pdf.
  • Import room loads via CSV file.
  • Web based.
  • Self host (run on your own infrastructure).
